Overview of HTML Style

With proper organization and layout formatting in HTML documents covered, let's look at how we can STYLE our HTML pages further.

Without the use of external CSS (cascading style sheets- used to format the layout of a webpage), the stylization capabilities for HTML alone are somewhat limited.

We will soon cover external CSS-- but not until after Assignment 1. So, what can we do in the meantime?

Fortunately, we CAN apply some style and design elements to our webpages by adding CSS formatting to our webpages in two ways-- no external CSS required!:

  • Inline CSS
  • Internal CSS

Inline and Internal CSS Design Possibilities

By using inline CSS and internal CSS without ANY external CSS, we can set:

  • Font family
  • Font size
  • Text color
  • Italic text
  • Bold text
  • Superscripts
  • Subscripts
  • Underline text
  • Line breaks
  • Line break points
  • Hyphenation
  • No line breaks
  • Text alignment
  • Background color
  • Background image
  • Many others!
