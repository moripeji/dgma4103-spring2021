Block and Inline Elements

There are two basic types of HTML elements:

    • Block-level Elements:
    • Take up space by default
    • Stack down the page
    • Take up as much horizontal space as possible
    • Push the next block-level element down
      • Examples:
      • Paragraphs
      • Ordered and unordered lists
      • All headings
      • Inline Elements:
      • Display in a line
      • Do not force the text after them into a new line
      • Can put several inline elements in a row
        • Examples:
        • Anchors (links)
        • Strong and emphasis (em)
        • Code


Why is this Important?

Knowing the difference between how block-level and inline elements work will help greatly with CSS layouts and further structuring our HTML code with divs, ids, classes, and spans for better CSS formatting ability.

Fundamental rule:
INLINE ELEMENTS CANNOT CONTAIN BLOCK-LEVEL ELEMENTS!
