Today's Task

After learning about HTML Layout, organization, and visual hierarchy, it's time to put your knowledge to work!

Edit your Assignment 1 sites to include the following features:

  1. HEADING SIZES
    • Demonstrate the proper use of heading sizes for greatest visual hierarchy
    • Use this page as your reference
  2. SEMANTIC STRUCTURAL ELEMENTS
    • Use tags such as header, main, footer, section, figure, etc. within the body of EACH of your Assignment 1 pages
    • Create a header with a main title for your entire site (h1 heading size), some one-liner text about your site (with p), and a nav that includes the links to the pages of your site (index, about page, Wordpress portfolio). Remember- this header needs to be copied and pasted onto EVERY one of your .html pages in the body!
    • Reposition the content of your pages to fit within different semantic structural elements. Use main, section, etc. EVERY page on your site should use this structure and organization!
    • Add CAPTIONS to your photo gallery pages (EACH ONE) by using the figure and figcaption elements
  3. NON-SEMANTIC STRUCTURAL ELEMENTS
    • Add div and span containers where you think could be necessary for later formatting
    • Make sure to include the "class" and/or "id" attributes in these div and span tags!
  4. FORMATTING/ ORGANIZATIONAL ELEMENTS
    • Add in some basic formatting/ organizational elements to your site to add some visual appeal AND to make your site easier for users to read and follow along with the content
    • Use tags for spacing like hr and br, as well as text formatting tags (u, i, strong, em, small, etc.) to do this
  5. NEED HELP?
    • Resources, reference guides, and tutorials have been posted on ALL of the above to Blackboard. Watch and read through if needed!
    • Professor Page's code is also LIVE and constantly updated via the link posted to Bb. Check out how the Assignment 1 Demo utilized these elements and features to structure the content- ask questions if needed!
